# 테스트
### 개발 방식

AI가 코드를 생성하는 방법은 여러 가지가 있습니다

1）직접 하나의 AI 창을 열고 동시에 프론트엔드 + 백엔드를 개발합니다 (“전체 프론트엔드와 백엔드를 구현하세요”)

2）두 개의 AI 창을 열고, 하나는 프론트엔드, 다른 하나는 백엔드입니다. 먼저 백엔드가 작업을 수행하고, 백엔드 개발이 완료되면 코드와 인터페이스 문서를 프론트엔드에게 전달한 후 개발을 진행합니다

3）두 개의 AI 창을 열고, 프론트엔드와 백엔드가 기술 사양 문서 기반으로 동시에 개발합니다

기업 표준의 개발 방식: 프론트엔드와 백엔드가 회의를 열고 합의; 백엔드가 인터페이스 문서를 제공; 그 후 프론트엔드와 백엔드가 동시에 개발

### 최종의 개발 방식

1）먼저 사양에 따라 백엔드가 인터페이스 문서를 생성합니다

2）프론트엔드는 인터페이스 문서에 따라 페이지를 개발합니다; 백엔드는 인터페이스 문서에 따라 구현합니다; 두者는 동시에 개발

### 백엔드 개발

AI에게 제공하는 프롬프트에 주의:

1. 먼저 사양에 따라 백엔드가 인터페이스 문서를 생성한 후, 이를 프론트엔드와 백엔드에게 동시에 개발하도록 합니다

2. 최신 기술 문서를 반드시获取해야 합니다

3. 무의미한 콘텐츠 생성을 방지
